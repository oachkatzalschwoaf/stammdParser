Dies ist eine quick and dirty Implementierung für ein spezifisches Problem.

Howto:
Einfach den entsprechenden export als csv im UTF-8 format speichern und mit dem Button unten auswählen.
Bei nicht UTF-8 formartierten dateien gehen die Umlaute verloren!
Die angepassten Daten werden anschließend in einer Tabelle angezeigt.

Quellcode siehe: Github
eingesetzte Libraries:
  • PapaParse
  • JQuery
  • JQuery Tablesorter
  • table2CSV
